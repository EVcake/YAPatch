# YAPatch

Yet Another Patching Tool for Android to load xposed modules

## Build
```shell
./gradlew buildAll
```

## 可用性

已测试可用的 应用-模块,[模块] 有

- QQ-QAuxiliary
- 哔哩哔哩-哔哩漫游
- 微信

不可用的 应用 有

- 菜鸟
- 知乎
